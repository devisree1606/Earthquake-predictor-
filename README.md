# Earthquake Prediction System

## Overview

The Earthquake Prediction System is a machine learning-based web application developed to predict the severity of earthquakes using earthquake-related parameters. The system analyzes input data and classifies earthquakes into different categories such as Low, Moderate, and Severe.

## Features

* User-friendly web interface using Streamlit
* Predicts earthquake severity instantly
* Simple and interactive design
* Machine learning-based prediction
* Fast and efficient performance

## Technologies Used

* Python
* Streamlit
* Pandas
* NumPy
* Scikit-learn

## How to Run

1. Install the required packages:

   ```
   pip install streamlit pandas numpy scikit-learn
   ```

2. Run the application:

   ```
   streamlit run app.py
   ```

3. Open the local URL displayed in the terminal.

## Project Workflow

1. Data Collection
2. Data Preprocessing
3. Model Training
4. Prediction
5. Result Display
